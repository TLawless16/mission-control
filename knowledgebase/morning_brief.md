I have planned out the exact architecture for the **Master SEO Auditor & Website Builder** you requested. Because this is a massive and powerful system, I have created a detailed project specification in Mission Control for your review: `knowledgebase/seo_agent_project.md`.

Here is the high-level summary of how the agent will operate when you are ready to build it today:

### 1. The Competitor Aggregation Engine
When given a primary keyword, the agent will tap into a SERP API (like DataForSEO) to find the top 20 ranking URLs. It will scrape all 20 sites, analyze their sitemaps, and group their content by page intent (e.g., all 20 Homepages together, all 20 Services pages together). 

### 2. The "Best-of-the-Best" Content Synthesis
Using an advanced LLM with a massive context window, the agent will synthesize all 20 Homepages into **one single master Homepage**. It will extract the best ranking factors while strictly applying your client's unique brand persona. 

### 3. The Page Ranking Evaluator
For non-standard pages (e.g., specific blog posts, hyper-niche service pages), the agent will score them on their SEO ranking value. If a page provides zero ranking juice, the agent will automatically discard it. If it is valuable, it will present a "Pros/Cons" recommendation list to you.

### 4. The Autonomous WordPress Builder
Once the master sitemap and content are approved, the agent will log into the WordPress site you provide. It will use the exact plugins you specified (Oxygen, Breakdance, Page Generator Pro) to physically assemble the pages. 

### 5. The Technical SEO Optimizer
Before finalizing, the agent will sweep the entire site to strictly enforce:
- **ADA Compliance** (Contrast ratios, ARIA labels).
- **JSON-LD Schema** (Every image, video, quote, and organization detail properly mapped).
- **Core Web Vitals** (Image compression, minimal DOM depth, alt tags on all media).
